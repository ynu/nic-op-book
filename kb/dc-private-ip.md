# 数据中心内网IP地址分配规则
本文描述数据中心内网IP地址的分配规则。

## 概述
为方便使用，数据中心使用10.0.0.0/8段作为私网地址。

## 地址段分配表
| *IPv4地址段*    |    *用途* |
| :-------- | :--------:|
| 10.0.0.1   | 保留，将来作为网关使用 |
| 10.0.0.2 - 10.0.0.255  | 物理服务器 |
| 10.0.1.0 - 10.0.1.255  | 其他物理设备 |
| 10.0.2.0 － 10.0.3.255 | 重要服务虚拟机 |
| 10.0.4.0 － 10.0.7.255 | 普通虚拟机 |
