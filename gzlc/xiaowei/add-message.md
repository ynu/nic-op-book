# 在素材库中添加素材

本文档描述如何在应用的素材库中添加素材。

## 前提

1. 已登录微信企业号后台。
2. 有管理应用的权限。

## 工作流程

1. 点击右侧“消息中心”菜单，进入消息中心。
2. 点击主窗口左上角的应用名称，选择需要添加素材的应用。
3. 选择主窗口上面中间位置的“素材库”选项卡。
4. 根据素材类型选择相应的选项卡。
5. 点击“新建素材”按钮开始创建素材。素材的具体创建过程及注意事项见下文。

### 新建“文字”素材

1. 在文本框中输入想要的文字。
2. 在右侧可预览文字效果。
3. 文本框中可使用部分HTML代码。包括：
	- 超链接。例如：`<a href="http://www.ynu.edu.cn">云大主页</a>`。

4. 编辑完成后，点击“保存”即可保存素材。

### 新建“图文”素材

1. 图文素材包括单图文消息和多图文消息。一般选择单图文消息，之后也可以变成多图文消息。
2. 按照要求填写标题、封面图片、作者、原文链接、摘要和正文。
3. 当需要将单图文消息变成多图文消息时，点击右侧预览图的“＋”按钮即可。
4. 编辑完成后，点击“保存”按钮即可保存素材。
